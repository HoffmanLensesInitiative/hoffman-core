## Kit Contents

### 1. Project Summary (1 page)

**What Hoffman Lenses Is:**
A browser-based research tool that makes behavioral manipulation visible to the people it operates on. It analyzes language patterns in real-time, identifying techniques platforms use to exploit psychological vulnerabilities for engagement.

**What It Is Not:**
- Not an ad blocker
- Not a privacy browser
- Not a censorship tool
- Not platform-specific

**Key Facts:**
- Built on Electron (Chromium-based, cross-platform)
- All analysis runs locally on device
- Zero data retention
- Open source (GitHub: HoffmanLensesInitiative/hoffman-core)
- Free for all users

**The Name:**
From the Hoffman Lenses in John Carpenter's 1988 film "They Live" — glasses that reveal hidden manipulation embedded in ordinary reality.

### 5. FAQ

**Q: Does Hoffman Lenses block content?**
A: No. It blocks nothing. It hides nothing. It makes visible. The user decides what to do with what they see.

**Q: Is this an AI tool?**
A: The browser includes a local language model (Llama 3.2, running entirely on-device) that analyzes text for manipulation patterns. No data is sent to external servers.

**Q: Is this a political tool?**
A: No. Hoffman Lenses identifies manipulation techniques regardless of political orientation. The same techniques appear across the political spectrum.

**Q: How is this different from fact-checking?**
A: Hoffman Lenses does not assess whether claims are true or false. It identifies *how* language is constructed to manipulate, regardless of the content's accuracy.

**Q: Is Hoffman Lenses affiliated with any organization?**
A: No. It is an independent project. It receives no funding from platforms, governments, or political organizations.

**Q: Can platforms block Hoffman Lenses?**
A: The browser reads rendered screen output — what the user actually sees. Platforms cannot obfuscate this without breaking their own product.
